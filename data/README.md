# Data Folder

This folder is used to store the dataset(s) needed for the analysis.

## Files

- `kickstarter_projects.csv`  
  Raw Kickstarter projects dataset downloaded from Kaggle:
  - Contains ~375k campaigns
  - Columns include:
    - `ID`, `Name`
    - `Category`, `Subcategory`
    - `Country`
    - `Launched`, `Deadline`
    - `Goal`, `Pledged`, `Backers`
    - `State` (e.g. `Successful`, `Failed`, `Canceled`, `Live`, `Suspended`)
